✨ Project Overview
Unathi’s Beauty Co is a front-end web project built to bridge the gap between professional beauty services and a sophisticated 
digital presence. The site features a "Radiant" gold-and-white theme, specifically engineered for high-speed performance and
elegant readability on mobile devices.

🎯 Key Features
Mobile-First Architecture: Designed for vertical scrolling with touch-friendly interaction points and 44px+ tap targets.

One-Tap Connectivity: Integrated tel: protocol and Google Maps deep-linking for instant calling and navigation.

Interactive Showroom: A space-saving mobile grid featuring "click-to-reveal" price lists using semantic HTML5 <details> and <summary> tags.

Smooth UX: Enhanced navigation using scroll-behavior: smooth and scroll-padding-top for a premium feel.

Full-Bleed Branding: Edge-to-edge color blocking for the Final CTA and Developer Attribution bar, providing a modern app-like aesthetic.

🛠️ Tech Stack
HTML5: Semantic structure (Header, Main, Section, Footer, Address).

CSS3: Flexbox & CSS Grid (Mobile-First approach), CSS Gradients, and custom transitions.

Google Fonts: Specialized font-pairing (Playfair Display & Montserrat) for a high-end boutique look.

Deployment: Hosted via Netlify with continuous integration.

📐 Mobile-First Design Decisions
Thumb-Zone Optimization: Strategic placement of the "Book Now" and "Call" actions for easy one-handed operation.

Privacy-First Location: Precise area-based location pinning (Bronkhorstspruit) to build trust while maintaining
professional exclusivity.

Performance: Zero-JS interactivity for UI elements, ensuring the site remains lightning-fast even on 3G/LTE mobile connections.
